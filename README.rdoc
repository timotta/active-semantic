= SemanticEntities

Experimental ruby gem to abstract sparql queries

== Example of use

Getting the uri:

  entity = SemanticEntity.find(@uri)
  puts entity.uri
  
Getting the an attribute:
  
  puts entity.name.first
  
Getting all attributes the same name:

  puts entity.site.each do |site|
    puts site
  end
